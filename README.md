## Get Started
This Project Need NodeJS 14x

### Cloning
```sh
> git clone https://github.com/FaizBastomi/kaguya-bot.git
> cd kaguya-bot
```
Install the dependencies:
```sh
> npm install
> npm update
```

### Run
```sh
> npm start
```
