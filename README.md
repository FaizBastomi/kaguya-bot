## Get Started

This Project Need NodeJS 18x or higher, [get here.](https://nodejs.org/en/download/current/)

### Cloning

```sh
> git clone https://github.com/FaizBastomi/kaguya-bot.git
> cd kaguya-bot
```

Install the dependencies:

```sh
> npm i
```

### Run

```sh
> npm start
```
